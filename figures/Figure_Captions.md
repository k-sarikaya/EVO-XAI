## Computers in Biology and Medicine Submission

**Figure 1.** Feature correlation matrix for the child dataset, showing moderate-to-high correlations among AQ-10 behavioral items (A1-A10).

**Figure 2.** EVO-XAI framework architecture showing the three-stage feature selection pipeline: Stage 1 (Chi-square filtering), Stage 2 (VIF-based multicollinearity detection), and Stage 3 (Evolutionary optimization).

**Figure 3.** MCC performance across VIF thresholds (5-13) for all three evolutionary algorithms and datasets. NSGA-II at VIF=13 achieves peak performance for adolescent data.

**Figure 4.** Comparative performance of evolutionary algorithms (NSGA-II, DE, Hybrid PSO+DE) across evaluation metrics (MCC, Sensitivity, Specificity) at VIF=13. Error bars represent standard deviation from 5-fold cross-validation.

**Figure 5.** SHAP-based feature importance analysis mapping selected features to DSM-5 diagnostic domains. Colors indicate DSM-5 criterion categories: Social Communication (blue), Relationships (green), Stereotyped Behaviors (red), Sensory (orange), Restricted Interests (purple).

**Figure 6.** Model robustness under Gaussian noise injection (σ = 0.05 to 0.20). All datasets retain >83% baseline MCC at σ=0.20, demonstrating stability against input perturbations.

**Figure 7.** Classifier comparison showing MCC and AUC performance across six machine learning algorithms. The stacking ensemble with evolutionary feature selection achieves competitive performance with enhanced interpretability.

**Figure 8.** Decision Curve Analysis showing net benefit across probability thresholds for EVO-XAI screening model compared to treat-all and treat-none strategies. The model provides clinical utility across the 10-40% threshold range.
